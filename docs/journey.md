# User Journey Map

| 단계 | 사용자 행동 | 감정 | Pain Point | 서비스 기회 |
|---|---|---|---|---|
| 쇼핑 시작 전 | 쇼핑앱 접속 및 상품 탐색 시작 | 무감각 | 무엇을 살지 명확하지 않음. | 사용자 취향 기반 스타일 추천 |
| 상품 탐색 | 여러 상품을 비교하며 탐색 | 피로 | 상품이 너무 많아 선택 어려움 | 개인화 상품 필터링 |
| 코디 고민 | 옷 조합을 직접 고민 | 귀찮음 | 코디를 스스로 구성해야 함 | AI 코디 자동 생성 |
| 구매 결정 | 장바구니에 담고 고민 | 망설임 | 확신 부족으로 구매 지연 | 완성된 스타일 제안 + 추천 이유 제공 |
| 구매 포기 / 완료 | 구매 또는 이탈 | 후회 / 만족 | 선택 피로로 이탈 발생 | 빠른 의사결정 유도 UX |
