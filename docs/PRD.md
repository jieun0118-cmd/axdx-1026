# PRD

## 문제정의
온라인 쇼핑 중 어떤 옷을 선택하고 코디할지 고민하는 상황

## 목표 사용자
20-30대 패션 초보 남성 소비자

## 핵심 가치 제안
사용자 체형·스타일을 기반으로 완성된 코디를 자동 추천하여 쇼핑 시간을 줄인다

## 핵심 시나리오
사용자가 자신의 체형·선호 스타일을 입력하면 AI가 개인 맞춤 코디를 생성하고 추천 상품과 함께 스타일을 제안한다

## MVP 기능 범위
1. 사용자 스타일/체형 입력
2. AI 코디 추천 (상의 + 하의 + 액세서리 등)
3. 추천 상품 리스트 제공
4. 코디 저장 (찜 기능)

## 제외할 기능
브랜드·입점 파트너 연동, 프리미엄 구독 및 결제 시스템, 커뮤니티(코디 공유·스타일 피드) 기능

## 주요 데이터
users, products, outfits, preferences

## KPI
- Recommendation Conversion Rate: 추천 코디 클릭 후 구매 수 / 추천 코디 클릭 수 / 목표 30%
- Outfit Completion Rate: 코디 확인 후 저장 또는 구매 수 / 코디 조회 수 / 목표 60%
- User Input Completion Rate: 스타일 정보 입력 완료 사용자 수 / 전체 사용자 수 / 목표 70%
- AI Recommendation Accuracy: 수정 없이 저장된 코디 수 / 전체 추천 코디 수 / 목표 80%
- Cart Abandonment Reduction: (기존 이탈률 - 적용 후 이탈률) / 기존 이탈률 / 목표 20% 감소

## 리스크
추천 정확도 부족 → 사용자 신뢰도 하락,
체형/스타일 입력의 주관성 문제,
상품 데이터 부족 시 추천 품질 저하,
초기 사용자 데이터 부족 (Cold Start 문제)

## 향후 확장 방향
이미지 기반 스타일 분석 (사진 업로드),
쇼핑몰 연동 (실제 구매 연결),
날씨/계절 기반 추천,
SNS 스타일 공유 기능,
AI 스타일 피드백 학습 (추천 고도화)
