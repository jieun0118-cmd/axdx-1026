# Acceptance Criteria

| ID | 기능 | Given | When | Then | 우선순위 |
|---|---|---|---|---|---|
| AC-01 | 사용자 정보 입력 | 스타일/체형 정보를 입력한 상태 | 저장 버튼을 클릭하면 | 사용자 정보가 저장된다 | High |
| AC-02 | 사용자 정보 입력 | 아무 값도 입력하지 않은 상태 | 저장 버튼을 확인하면 | 버튼이 비활성화된다 | High |
| AC-03 | 사용자 정보 입력 | 공백만 입력한 상태 | 저장 버튼을 확인하면 | 버튼이 비활성화된다 | High |
| AC-04 | 사용자 정보 입력 | 입력값이 허용 범위를 초과한 상태 | 입력창을 확인하면 | 입력이 제한된다. | Medium |
| AC-05 | 코디 추천 | 사용자 정보가 저장된 상태 | 추천 버튼을 클릭하면 | 맞춤 코디가 생성된다 | High |
| AC-06 | 코디 추천 | 사용자 정보가 없는 상태 | 추천 버튼을 클릭하면 | 입력 요청 메시지가 표시된다 | High |
| AC-07 | 상품 추천 | 코디가 생성된 상태 | 추천 결과를 확인하면 | 코디에 포함된 상품 목록이 표시된다 | High |
| AC-08 | 코디 저장 | 코디 추천 결과가 있는 상태 | 저장 버튼을 클릭하면 | 코디가 저장된다 | Medium |
| AC-09 | 코디 저장 | 저장된 코디가 있는 상태 | 저장 목록을 확인하면 | 저장된 코디 리스트가 표시된다 | Medium |
| AC-10 | 추천 품질 | 추천 결과가 생성된 상태 | 사용자가 결과를 확인하면 | 추천 이유 또는 스타일 설명이 제공된다 | Low |
