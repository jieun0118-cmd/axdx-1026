# Problem Definition

## 1. 주제
AI 기반 개인 맞춤형 쇼핑 추천 서비스

## 2. 문제 상황
온라인 쇼핑 중 어떤 옷을 선택하고 코디할지 고민하는 상황

## 3. 핵심 사용자
20-30대 패션 초보 남성 소비자

## 4. 사용자가 겪는 불편
상품은 많지만 체형·스타일에 맞는 선택이 어려움,
개별 상품은 찾을 수 있어도 전체 코디 완성이 어려움,
구매 결정까지 시간이 오래 걸리고 피로감 증가

## 5. 현재 대안과 한계
무신사, 지그재그, 에이블리 등 기존 플랫폼은 상품 추천 중심으로 운영되며, 코디 완성 기능이 부족하고 개인화 추천에 한계가 존재함. 획일적인 추천으로 사용자 만족도와 추천 신뢰도가 낮으며, 체형·사이즈·핏 정보 부족으로 구매 후 반품·교환이 빈번하게 발생함.

## 6. 서비스 아이디어
사용자의 클릭·구매·선호 스타일 데이터를 AI가 분석하여 체형과 취향에 맞는 상품을 추천하고, 상의·하의·신발 등을 조합한 완성형 코디 세트를 제공함. 데이트, 출근, 여행 등 상황별 스타일링 추천도 지원하는 AI 개인 스타일 코치 서비스.

## 7. 4주 안에 구현 가능한 최소 기능
사용자 체형·선호 스타일 프로필 입력,
추천 코디 목록 조회(5~10개),
상품별 가격·사이즈·구매 버튼 표시,
프로필 미입력 시 안내 메시지 처리

## 8. 데이터로 검증할 수 있는 지표
추천 상품 클릭률, 사용자 체류 시간, 구매 전환율, 재방문율

## 9. 구현 범위에서 제외할 것
브랜드·입점 파트너 연동, 프리미엄 구독 및 결제 시스템, 커뮤니티(코디 공유·스타일 피드) 기능
